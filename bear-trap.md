

# 🐻 BEAR TRAP GUIDE 🐻

**What is Bear Trap?**
Bear Trap is an alliance event where members work together to defeat the Bear by hosting and joining rallies. The more damage the alliance deals, the better the rewards for everyone.

## 🎁 Rewards

* Personal rewards based on the damage you deal.
* Alliance milestone rewards as the alliance reaches higher Bear levels.
* Better performance = Better chests and more rewards for everyone.

## ⚔️ Rally Roles

**R5 / R4 / R3**

* Start (host) rallies.
* Hosting rallies gives more personal points since your strongest heroes lead the attack.
* More rallies = More total alliance damage = Better rewards for everyone.

**R2 / R1**

* Join rallies started by R3, R4, and R5.
* Help fill rallies quickly to maximize damage.

## 🐻 Bear Trap Rules

### 🔵 Rally Host

* Use your **best 3 heroes**.
* **No blue heroes.**

### 🔵 Rally Joiners

Use **ONE** of these as your first hero:

* Amadeus + Hero + Hero
* Chenko + Hero + Hero
* Yeonwoo + Hero + Hero
* Amane + Hero + Hero

If you don't have any of these heroes, **join without heroes**.

### 🔴 Important

* Rally hosts should remove members using the wrong heroes.
* Joiners can simply rejoin with the correct setup.

💙 Let's work together, fill rallies quickly, and maximize everyone's rewards!
